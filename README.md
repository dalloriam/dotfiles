# Dotfiles
Miscellaneous images &amp; images to make my life easier.

## Setting Up
```shell
[~] > git clone git@github.com:dalloriam/dotfiles.git .dotfiles
[~] > cd .dotfiles
[~/.dotfiles] make all
```

## Requirements
* Fish shell
* Nu shell
    * `cargo install nu` on linux
    * `brew install nushell` on mac
    * `winget install nushell` on windows
